0. Старт (1–2 занятия)
- Настройка MCP на Claude, как правильно работать с AI-ассистентом
- IDE и среда: Go + VSCode/GoLand, базовые плагины
- Как работают языки: компиляция vs интерпретация, что происходит от кода до бинарника
- Почему Go: статическая типизация, горутины, где используется в индустрии

1. Основы Go (1–1.5 мес)
- переменные, типы данных, условия, циклы, функции (35%)
- структуры данных, указатели (20%)
- структуры, методы, интерфейсы, модули (25%)
- работа с файлами, обработка ошибок, json, базовое тестирование (20%)

2. Concurrency в Go (3–4 нед)
- горутины, каналы
- worker pool, WaitGroup, Mutex
- context.Context — отмена, таймауты
- практика с конкурентной обработкой данных

3. Web на Go (2 месяца)
- HTTP-основы, net/http, потом Gin. Роутинг, middleware
- Работа с PostgreSQL (sqlx/pgx), миграции
- REST API и структура веб проекта (layered architecture)
- Очереди: Redis и/или RabbitMQ

4. Продвинутый Backend (2 месяца)
- gRPC и protobuf: написание сервисов, стриминг, взаимодействие между сервисами
- Аутентификация и безопасность: JWT, refresh токены, middleware авторизации
- Паттерны микросервисной архитектуры: saga, circuit breaker, retry, идемпотентность
- Продвинутая работа с БД: индексы, транзакции, оптимизация запросов, connection pool
- Базовый Docker — только чтобы уверенно запускать окружение локально
