# Go Lessons

Авторский курс по Go, разбитый по блокам занятий. Каждый блок — отдельная папка: теория прозой в `README.md`, практика в `.go`-файлах с заглушками, решения — в подпапке `resolve/`.

## Структура

```
go-lessons/
├── l1/   — основы: переменные, типы, ввод-вывод
├── l2/   — условия, циклы и функции
├── l3/   — условия, циклы, рекурсия (практика)
├── l4/   — срезы, указатели, структуры, JSON
└── l5/   — срезы вглубь, интерфейсы, обработка ошибок
```

## Как запускать

Из папки нужного урока:

```bash
go run .
```

В `main.go` раскомментируй вызов нужного блока.
